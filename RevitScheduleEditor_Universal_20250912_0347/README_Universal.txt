UNIVERSAL REVIT SCHEDULE EDITOR
================================

COMPATIBILITY:
- Revit 2020, 2021, 2022, 2023, 2024, 2025, 2026
- Windows 10/11
- .NET Framework 4.8

INSTALLATION:
1. Run "Install_Universal.bat" as Administrator
2. The installer will detect all installed Revit versions
3. Add-in will be installed for each found version
4. Restart Revit to see the new command
